## Structure
- index.html — full skeleton in place
- style.css — empty, ready for styling
- app.js — empty, ready for JavaScript

## HTML sections
- header — app title
- #search-section — text input and search button
- #error-message — hidden by default, shown on API errors
- #profile-section — hidden by default, revealed after a successful search

  - #languages-section — top languages with visual chart
  - #repos-section — repo list with starred/recently updated toggle
- footer — API credit
